# Raw Data Directory

This directory contains raw datasets that are not committed to git due to their size.

## MovieLens 32M Dataset

**Location:** `ml-32m/`

**Download:** https://grouplens.org/datasets/movielens/32m/

**Files:**
- `movies.csv` - Movie titles and genres (4 MB)
- `links.csv` - Links to TMDb and IMDb (1 MB)
- `ratings.csv` - User ratings (836 MB)
- `tags.csv` - User-generated tags (69 MB)

**Instructions:**
1. Download `ml-32m.zip` from the link above
2. Extract to `backend/raw/ml-32m/`
3. Verify files exist:
   ```bash
   ls -lh backend/raw/ml-32m/
   # Should show: movies.csv, links.csv, ratings.csv, tags.csv, README.txt
   ```

## TMDb Metadata (Optional)

**Location:** `tmdb/`

**Generated by:** `backend/scripts/fetch_tmdb_metadata.py`

**Files:**
- `tmdb_metadata.csv` - Enriched movie metadata from TMDb API

**Instructions:**
1. Get API token from https://www.themoviedb.org/settings/api
2. Create `.env` file: `TMDB_BEARER_TOKEN=your_token`
3. Run: `python -m scripts.fetch_tmdb_metadata`
4. Output will be saved to `raw/tmdb/tmdb_metadata.csv`
